# EA Risk Calculator (Excel)

Spreadsheet templates for correlation-aware risk management, trend filter, and simulation study.

**Start here:** [EXCEL-SETUP.md](EXCEL-SETUP.md)

## CSV files (`csv/`)

| File | Use |
|---|---|
| `01-settings.csv` | Account limits |
| `02-trend-table.csv` | Trend filter (ON) |
| `03-pip-values.csv` | Broker pip values |
| `04-positions-template.csv` | Live open positions (blank) |
| `05-sniper-inputs-template.csv` | New trade calculator map |
| `06-sim-open-positions.csv` | Simulation tab — fake open trades |
| `07-sim-test-candidates.csv` | 8 study scenarios (T1–T8) |

Import into Excel → two tabs: **Risk** (live) + **Sim** (practice).
