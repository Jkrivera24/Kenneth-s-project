## Daily routine automation

The **routine-automation** app helps you run and automate a daily schedule from your phone:

- Web dashboard for morning / work / evening routines  
- QR scanner to complete steps from your phone  
- iOS Shortcuts and Android Tasker webhooks  
- JSON import for custom schedules  

See [routine-automation/README.md](./routine-automation/README.md) to run it.
