# BOLLFILTER — Manual Cleaning of Candle Filter Elements

**Manual reference:** Section 13  
**System:** Protection Systems / automatic filter (candle elements)  
**Use:** Planned maintenance only — follow ship-specific SMS and OEM manual.

---

## Before you start

| Item | Done |
|------|:----:|
| Work permit / risk assessment completed (hot oil, solvents, compressed air) | ☐ |
| Correct manual and filter type confirmed on board | ☐ |
| Solvent selected and SDS available (see §13.2) | ☐ |
| Dedicated soak tank and **separate** rinse tank for §13.5 | ☐ |
| BOLL cleaning gun available for compressed air blow-through | ☐ |
| Suitable drain stand (e.g. perforated sheet) | ☐ |
| PPE: gloves, goggles, apron; ventilation where solvents used | ☐ |

---

## Critical warning — drain before dismantling

> **Before cartridge elements are dismantled**, the automatic filter must be **completely drained** by **automatic back-flushing of all filter chambers**.
>
> Use **manual activation** on the **control box**.  
> Confirm **liquid level is below the cartridge element** before dismantling.

| Step | Action | Done |
|------|--------|:----:|
| W1 | Initiate manual backflush from control box — **all chambers** | ☐ |
| W2 | Verify filter is fully drained | ☐ |
| W3 | Confirm liquid level **below** cartridge element | ☐ |
| W4 | Only then proceed to remove filter element assembly | ☐ |

## Procedure checklist

### 13.1 — Soak (contaminants drain downward)

| Step | Action | Done |
|------|--------|:----:|
| 1 | Remove **whole filter element assembly** | ☐ |
| 2 | Place assembly in tank with **candle openings facing down** | ☐ |
| 3 | Fill tank with approved solvent (see table below) | ☐ |
| 4 | Allow detached contaminants to sink out of candles | ☐ |

### 13.2 — Soaking time and solvent (do not exceed)

| Option | Solvent | **Maximum soaking time** | Selected |
|--------|---------|--------------------------|:--------:|
| a | BOLL CLEAN 2000 (cold) | **24 hours** | ☐ |
| b | Filterclean (Vecom) | **12 hours** | ☐ |
| c | Reiniger B85 (Vecom) | **12 hours** | ☐ |
| d | Gas oil | **48 hours** | ☐ |

**Soak start:** _______________ **Soak end:** _______________

### 13.3 — Drain solvent

| Step | Action | Done |
|------|--------|:----:|
| 5 | Remove whole filter element assembly from tank | ☐ |
| 6 | Place on drain stand with **candle openings pointing down** | ☐ |
| 7 | Allow solvent to drain off | ☐ |

### 13.4 — First compressed-air blow

| Step | Action | Done |
|------|--------|:----:|
| 8 | Using **cleaning gun supplied**, blow compressed air **inside → outside** through each candle | ☐ |

### 13.5 — Rinse in **fresh** cleaner (separate tank only)

> **Warning:** Rinse only in a **separate tank** with **clean** solvent. Used rinse solvent may be reused for the **next soaking** cycle only (not for final rinse).

| Step | Action | Done |
|------|--------|:----:|
| 9 | Immerse complete assembly in **fresh** cleaner, candles **down** | ☐ |
| 10 | Rinse with **up and down motion** through candles | ☐ |

### 13.6 — Final drain and dry

| Step | Action | Done |
|------|--------|:----:|
| 11 | Drain assembly again (candles down) | ☐ |
| 12 | Dry by blowing compressed air **inside → outside** again | ☐ |

**Expected result (manual method per OEM):** ~**60% clean** — adequate for similar applications.

### 13.7 — Higher cleanliness (optional)

| Step | Action | Done |
|------|--------|:----:|
| 13 | For ~**100%** cleaning: use **Type 5.04 High Pressure Cleaning Unit** with **BOLL CLEAN 2000** | ☐ |
| 14 | Refer separate document: *Filter Cleaning Unit Type 5.04* | ☐ |

## After cleaning — before return to service

| Item | Done |
|------|:----:|
| Inspect candles and seals for damage; replace per spare parts list | ☐ |
| Reinstall per OEM assembly procedure (reverse of removal) | ☐ |
| Concluding work: fill, vent, leak check per manual | ☐ |
| Return filter to service; monitor ΔP and backflush function | ☐ |
