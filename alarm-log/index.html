Send to GitHub

Your alarms are saved on this phone. Tap below to download a file, then upload it to GitHub (laptop is easier).

File name: incident-log.csv

Add to Home Screen (like an app)

  1. Open this page in Chrome on your phone.
  2. Menu ⋮ → Add to Home screen or Install app.
  3. Open from the new icon — works offline at sea.

Upload CSV to GitHub (laptop)

  1. Tap Sync → Download CSV for GitHub.
  2. On laptop: open your repo → alarms/data/incident-log.csv.
  3. Click pencil → paste file contents → Commit.
  4. Status DONE moves alarm to resolved/icms/ on GitHub.

Status labels

IN PROGRESS — alarm active
CLEARED — gone, no reset
RESET — reset on panel
DONE — closed and verified
RECURRING — came back again
